# Café Luna – Local Café Website

## Overview
This starter project is part of  ACIT 1620 – Fundamental Web Technologies.  
Create a cozy, welcoming café website with HTML and CSS.

## File Structure
```
CafeLuna_Project/
├── index.html
├── menu.html
├── about.html
├── css/
│   └── style.css
└── images/
```

## Setup
1. Extract the ZIP file.
2. Open in VS Code.
3. Edit each HTML page and `css/style.css`.
4. Add café photos in the images folder.
5. Test responsiveness on desktop and mobile.
